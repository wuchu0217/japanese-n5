もんだい１　よみかた

＿＿のことばは どうよみますか。１・２・３・４から いちばんいいものを ひとつ えらんでください。

（1）今日は天気がいいですね。

✅ 正解：1. てんき (tenki) ＝ 天氣
2. でんき (denki) ＝ 電氣　3. てっき (tekki) ＝ 鐵器　4. てんぎ (tengi) ＝ 無此詞

（2）雪が降っています。

✅ 正解：1. ゆき (yuki) ＝ 雪
2. あめ (ame) ＝ 雨　3. くも (kumo) ＝ 雲　4. かぜ (kaze) ＝ 風

（3）今日は暑いですね。

✅ 正解：1. あつい (atsui) ＝ 炎熱的
2. さむい (samui) ＝ 寒冷的　3. すずしい (suzushii) ＝ 涼爽的　4. あたたかい (atatakai) ＝ 溫暖的

もんだい２　文法穴埋め

（　）に なにを いれますか。１・２・３・４から いちばんいいものを ひとつ えらんでください。

（4）今日は（　）が降っています。

✅ 正解：1. あめ (ame) ＝ 雨（「降る」と組み合わせて使う）
2. かぜ (kaze) ＝ 風　3. てんき (tenki) ＝ 天氣　4. はれ (hare) ＝ 晴天

（5）外は（　）から、上着を着てください。

✅ 正解：1. さむい (samui) ＝ 寒冷，所以需要穿上衣
2. あつい (atsui) ＝ 炎熱　3. すずしい (suzushii) ＝ 涼爽　4. あたたかい (atatakai) ＝ 溫暖

（6）今日は（　）晴れたり曇ったりします。

✅ 正解：1. ときどき (tokidoki) ＝ 有時候
2. いつも (itsumo) ＝ 總是　3. ぜんぜん (zenzen) ＝ 完全不　4. もっと (motto) ＝ 更加

もんだい３　場面

★の ひとは なんと いいますか。１・２・３・４から いちばんいいものを ひとつ えらんでください。

（7）★友達に今日の天気を聞きたいです。何と言いますか？

✅ 正解：1. Kyō no tenki wa dō desu ka? ＝ 今天天氣怎麼樣？
2. (tenki ga suki desu) 我喜歡天氣　3. (ame ga furimasu) 會下雨　4. (harete kudasai) 請放晴（不自然）

（8）★外が寒いです。友達にコートを持ってくるように伝えたいです。

✅ 正解：1. Soto wa samui desu kara, kōto wo motte kite kudasai. ＝ 外面很冷，請帶外套來。
2. (kōto wa doko desu ka) 外套在哪裡？　3. (samuku nai desu) 不冷　4. (kōto ga suki desu) 我喜歡外套

（9）★今日は晴れています。嬉しいです。何と言いますか？

✅ 正解：1. Kyō wa harete ite, ii desu ne! ＝ 今天放晴，真好！
2. (kyō wa ame desu ne) 今天下雨呢　3. (tenki ga suki janai desu) 我不喜歡天氣　4. (kumori desu ne) 是陰天呢

もんだい４　語彙

＿＿のことばに いちばん ちかい いみの ものを １・２・３・４から ひとつ えらんでください。

（10）今日の外はさむいです。

✅ 正解：1. 体が冷たく感じます (karada ga tsumetaku kanjimasu) ＝ 身體感覺冷冷的
「さむい」= 寒冷，與「冷たく感じる」最接近
2. (atsuku) 熱的　3. (suzushiku) 涼爽的　4. (atatakaku) 溫暖的
