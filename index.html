📝 試験モード（N5 形式）

もんだい１　よみかた — 漢字の読み方を選んでください
1. スーパーで　肉を　かいました。
📌 niku（肉 = meat / 肉）
1. にく = niku / 肉 ✓　2. みず = mizu / 水　3. さかな = sakana / 魚　4. やさい = yasai / 蔬菜
2. これは　野菜と　くだものです。
📌 yasai（野菜 = vegetables / 蔬菜）
1. やかい ✗　2. のさい ✗　3. やさい = yasai / 蔬菜 ✓　4. わかい = wakai / 年輕
3. きのう　魚を　たべました。
📌 sakana（魚 = fish / 魚）
1. にく = niku / 肉　2. さかな = sakana / 魚 ✓　3. みず = mizu / 水　4. もり = mori / 森林
もんだい２　文法穴埋め — （　）に入るものを選んでください
4. わたしは　まいにち　ごはんを　（　　）。
📌 食べる = taberu / 吃（to eat）
1. のみます = nomimasu / 喝　2. たべます = tabemasu / 吃 ✓　3. みます = mimasu / 看　4. かきます = kakimasu / 寫
5. この　パンは　（　　）です。
📌 おいしい = oishii / 好吃（delicious）
1. おいしい = oishii / 好吃 ✓　2. たかい = takai / 貴　3. やすい = yasui / 便宜　4. あたらしい = atarashii / 新
6. れいぞうこに　やさいが　（　　）　あります。
📌 たくさん = takusan / 很多（a lot）
1. ちょっと = chotto / 一點點　2. たくさん = takusan / 很多 ✓　3. すこし = sukoshi / 少許　4. あまり = amari / 不太…
もんだい３　場面 — ★の人は何と言いますか
7. レストランで、ともだちに「この魚はどう？」ときかれました。食べてみたら、とてもおいしかったです。★あなたは何と言いますか？
📌 食べてみたら→とてもおいしかった → おいしいです！
1. おいしいです = oishii desu / 好吃！✓　2. からいです = karai desu / 辣　3. つめたいです = tsumetai desu / 冷　4. いいです = ii desu / 不用了
8. スーパーで　たまねぎ・にんじん・じゃがいも・キャベツを　かいました。★あなたは　何を　かいましたか？
📌 すべて野菜（蔬菜）です。yasai = 野菜 ✓
1. くだもの = kudamono / 水果　2. やさい = yasai / 蔬菜 ✓　3. さかな = sakana / 魚　4. にく = niku / 肉
9. あなたは　のどが　かわきました。★何と言いますか？
📌 のどがかわく = thirsty → 水をください！
1. ご飯をください = gohan wo kudasai / 請給我飯　2. にくをたべたい / 我想吃肉　3. みずをください = mizu wo kudasai / 請給我水 ✓　4. パンをください / 請給我麵包
もんだい４　語彙 — いちばん近い意味を選んでください
