もんだい１　よみかた（漢字の読み方）

１．読書 のよみかたとして正しいものはどれですか。

読書 = どくしょ (doku-sho) = 讀書
a) どくしょ (doku-sho) ✓  b) よむしょ (yomu-sho) ✗  c) よみしょ (yomi-sho) ✗  d) どくしゅ (doku-shu) ✗

２．働く のよみかたとして正しいものはどれですか。

働く = はたらく (hataraku) = 工作
a) うごく (ugoku) 動く ✗  b) はたらく (hataraku) ✓  c) あるく (aruku) 歩く ✗  d) はしる (hashiru) 走る ✗

３．出口 のよみかたとして正しいものはどれですか。

出口 = でぐち (de-guchi) = 出口 / exit
a) でぐち (de-guchi) ✓  b) でくち ✗  c) しゅつぐち ✗  d) しゅっこう ✗

もんだい２　文法穴埋め（（　）に入るものをえらんでください）

４．まいにち、8じに（　）。

毎日の習慣は丁寧形「〜ます」で表します。
a) 起きます (okimasu) ✓ 我每天8點起床
b) 起き (oki) ✗ 不完整  c) 起きる (okiru) ✗ 普通形  d) 起きた (okita) ✗ 過去形

５．図書館で本を（　）のが好きです。

「〜のが好き」の前は動詞の辞書形を使います。
a) 読む (yomu) ✓ 我喜歡在圖書館讀書
b) 読み ✗  c) 読んだ ✗ 過去形  d) 読みます ✗ 文法的に不自然

６．友達と公園で（　）ました。

「〜ました」の前は連用形（ます形の語幹）。遊ぶ → 遊び＋ました
a) 遊ぶ ✗ 辞書形  b) 遊び (asobi) ✓ 我們在公園玩了
c) 遊んで ✗ て形  d) 遊ぼう ✗ 意志形

もんだい３　場面（★の人は何といいますか）

７．★ 友達から本を借りたいです。友達に何といいますか。

本を借りる＝「貸してください」と頼む表現。
b) 貸してください (kashite kudasai) ✓ 請借給我這本書
a) ✗ 質問のみ  c) ✗ 場所を聞く  d) ✗ 存在確認

８．★ 授業の後で家に帰りたいです。先生に何といいますか。

許可を求めるには「〜てもいいですか」を使います。
c) 帰ってもいいですか (kaette mo ii desu ka) ✓ 下課後可以回家嗎？
a) ✗ 命令  b) ✗ 質問のみ  d) ✗ 宣言（失礼）

９．★ 友達を公園に誘いたいです。友達に何といいますか。

誘う表現＝「〜ませんか」（一起～好嗎？）
a) 遊びませんか (asobimasen ka) ✓ 要不要一起去公園玩？
b) ✗ 過去の質問  c) ✗ 公園で働くのは変  d) ✗ 現在進行

もんだい４　語彙（いちばん近い意味のことば）

１０．「はたらく」とほぼ同じ意味のことばはどれですか。

働く (hataraku) ≒ 仕事をする (shigoto wo suru) = 工作
a) あそぶ (asobu) 玩耍 ✗  b) ねる (neru) 睡覺 ✗
c) しごとをする ✓ 工作  d) でかける (dekakeru) 出門 ✗
