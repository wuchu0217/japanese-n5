もんだい１　よみかた（読み方）
１. ＿＿の　ことばの　よみかたは　どれですか。
「頭が　いたいです。」
頭 = atama（あたま）= 頭部
１ あし = ashi = 腳／腿　２ あたま = atama = 頭部 ✓　３ かお = kao = 臉　４ くち = kuchi = 嘴巴
２. ＿＿の　ことばの　よみかたは　どれですか。
「元気ですか。」
元気 = genki（げんき）= 精神好
１ kenki ✗　２ keiki ✗　３ げんき = genki = 精神好 ✓　４ geiki ✗
３. ＿＿の　ことばの　よみかたは　どれですか。
「足が　いたいです。」
足 = ashi（あし）= 腳／腿
１ て = te = 手　２ くち = kuchi = 嘴巴　３ め = me = 眼睛　４ あし = ashi = 腳／腿 ✓
もんだい２　ぶんぽう（文法）
４. （　）に　なにを　いれますか。
「頭（　）いたいです。」
形容詞文の主語には「が」を使います。（Subject + ga + adjective）
１ を = wo（目的語）　２ が = ga（主語）✓　３ に = ni（方向・時間）　４ で = de（手段・場所）
５. （　）に　なにを　いれますか。
「毎朝、顔（　）洗います。」
動作の目的語は「を」を使います。（Verb object = wo）
１ に = ni　２ で = de　３ を = wo（目的語）✓　４ が = ga
６. （　）に　なにを　いれますか。
「彼女の目（　）大きいです。」
話題を提示するときは「は」を使います。（Topic marker = wa）
１ は = wa（話題）✓　２ が = ga（主語）　３ を = wo（目的語）　４ に = ni
もんだい３　ばめん（場面）
７. ★の　ひとは　なんと　いいますか。
😷 Aさんは　頭が　いたいです。Bさんに　はなします。
★Aさんは　なんと　いいますか。
体の不調を伝える表現。（Expressing physical discomfort）
１ Genki desu ka?（元気？）　２ Atama ga itai desu.（頭痛）✓　３ Ashi ga itai desu.（腳痛）　４ Arigatou gozaimasu.（謝謝）
８. ★の　ひとは　なんと　いいますか。
🤝 Aさんは　しばらく　あっていない　ともだちに　あいました。
★Aさんは　なんと　いいますか。
久しぶりの再会の挨拶。（Greeting after not seeing someone for a while）
１ sumimasen（打擾了）　２ arigatou（謝謝）　３ Genki desu ka?（你好嗎？）✓　４ gomen nasai（對不起）
９. ★の　ひとは　なんと　いいますか。
🎬 Aさんは　えいがを　みています。Bさんは「なにをしていますか」と　ききます。
★Aさんは　なんと　こたえますか。
「見る」= miru = 看。映画を見る = to watch a movie.
１ きく = kiku（聽）　２ みる = miru（看）✓　３ たべる = taberu（吃）　４ のむ = nomu（喝）
もんだい４　ごい（語彙）
１０. ＿＿に　いちばん　ちかい　いみの　ことばは　どれですか。
「映画を　みます。」
「見る」≒「眺める」— どちらも視覚的な行為です。（Both involve visual perception）
１ kikimasu = 聽　２ tabemasu = 吃　３ nagamemasu = 凝視・看 ✓　４ hashirimasu = 跑
