田中さんは毎日、朝六時に起きます。今日は日曜日です。でも、田中さんはもう起きています。外はまだ早い時間ですが、田中さんは朝ごはんを食べます。昼は友達と一緒に公園へ行きます。夜は家でゆっくり休みます。

聽力問題

もんだい１　よみかた

＿＿のことばの読み方として、一番いいものを１・２・３・４から一つ選んでください。

（１）毎日、朝六時に起きます。
ローマ字：asa（朝）/ hiru（昼）/ yoru（夜）/ yūgata（夕方）
中文：早上 / 中午 / 晚上 / 傍晚
（２）今日は月曜日です。
ローマ字：getsuyōbi / kayōbi / suiyōbi / mokuyōbi
中文：星期一 / 星期二 / 星期三 / 星期四
（３）私は早い時間に起きます。
ローマ字：osoi / takai / hayai / ōi
中文：晚・慢 / 高・貴 / 早・快 / 多い

もんだい２　ぶんぽう（文法）

（　）に入れるのに一番いいものを、１・２・３・４から一つ選んでください。

（１）毎朝六時（　）起きます。
中文：每天早上六點起床。時間點後用助詞「に」
選項：に(時間助詞✓) / で(場所・手段) / を(受詞) / が(主語)
（２）今日（　）日曜日です。
中文：今天是星期日。主題助詞「は」用於主題說明
選項：が(強調) / は(主題✓) / を(受詞) / の(所有)
（３）もう夜（　）なりました。
中文：已經變成晚上了。「〜になる」固定搭配「に」
選項：が / を / に(〜になる✓) / は

もんだい３　ばめん（場面）

★の人は何と言いますか。一番いいものを１・２・３・４から一つ選んでください。

（１）月曜日の朝、学校で先生に会いました。★の学生は何と言いますか。
情境：星期一早上在學校遇到老師。
選項：早安✓ / 午安 / 晚安 / 再見
（２）夜、友達と別れます。★の人は何と言いますか。
情境：晚上和朋友道別。
選項：早安 / 晚安道別✓ / 謝謝 / 不好意思
（３）今、何時か知りたいです。★の人は何と言いますか。
情境：想知道現在幾點。
選項：今天星期幾？/ 現在幾點？✓ / 今天幾號？/ 現在在哪裡？

もんだい４　ごい（語彙）

＿＿のことばに意味が一番近いものを、１・２・３・４から一つ選んでください。

（１）もうごはんを食べましたか。
「もう」= 已經（表示動作已完成）
ローマ字：sudeni / mada / itsumo / tokidoki
中文：已經✓ / 還沒 / 總是 / 有時
