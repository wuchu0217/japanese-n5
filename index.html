もんだい１ よみかた

＿＿のことばはどう よみますか。1〜4から いちばん いい ものを えらんでください。

1. 机の 上に ノートが あります。

ヒント：desk／書桌 (shū zhuō)
1. いす (i-su) 椅子 ｜ 2. つくえ (tsu-ku-e) 書桌 ✓ ｜ 3. たな (ta-na) 架子 ｜ 4. おく (o-ku) 放置

2. このベッドは 重い です。

ヒント：heavy／重的 (zhòng de)
1. はやい (ha-ya-i) 快 ｜ 2. たかい (ta-ka-i) 高／貴 ｜ 3. おもい (o-mo-i) 重 ✓ ｜ 4. おおい (o-o-i) 多

3. 棚に 本を 置きます。

ヒント：shelf／架子 (jià zi)
1. つくえ 書桌 ｜ 2. たな (ta-na) 架子 ✓ ｜ 3. いす 椅子 ｜ 4. まど (ma-do) 窗戶

もんだい２ 文法穴埋め

（　）に 何を 入れますか。1〜4から いちばん いい ものを えらんでください。

4. 食べ物を れいぞうこ（　）入れます。

ヒント：「〜に入れる」= put into〜／放入〜 (fàng rù)
1. を 受詞 ｜ 2. が 主語 ｜ 3. に 方向・場所 ✓ ｜ 4. で 手段

5. （　）新しい椅子を 買いました。

ヒント：「もう〜ました」= already〜／已經 (yǐ jīng)
1. もう (mo-u) 已經 ✓ ｜ 2. まだ (ma-da) 還沒 ｜ 3. また (ma-ta) 又 ｜ 4. いつも (i-tsu-mo) 總是

6. この テーブルを 毎日 （　）います。

ヒント：「使う」= to use／使用 (shǐ yòng)
1. おいて 放置 ｜ 2. つかって (tsu-ka-t-te) 使用 ✓ ｜ 3. みて 看 ｜ 4. きいて 聽

もんだい３ 場面

★の ひとは なんと いいますか。1〜4から いちばん いい ものを えらんでください。

7. 部屋に 入ります。椅子に 座りたいです。友だちに 頼みます。★

ヒント：「〜てください」= Please〜／請〜 (qǐng)
2. 持ってきてください (mot-te-ki-te ku-da-sa-i) 請帶來 ✓

8. 机の 上に 荷物を 置きたい。友だちに 許可を もらいます。★

ヒント：「〜てもいいですか」= May I〜？／可以〜嗎？(kě yǐ ... ma)
2. 机の上に置いてもいいですか ✓ 可以放在書桌上嗎？

9. 友だちの 冷蔵庫は とても 大きいです。すごいと 思います。★

ヒント：「〜ですね」同感表現／〜呢 (ne)
3. 冷蔵庫が大きいですね (re-i-zo-u-ko ga o-o-ki-i de-su ne) 冰箱很大呢 ✓

もんだい４ 語彙

＿＿のことばと いちばん ちかい いみの ものを 1〜4から えらんでください。

10. 荷物を 棚の 上に 置きました。

ヒント：「置く」≒「のせる」= to place on／放上去 (fàng shàng qù)
1. もっていく 帶走 ｜ 2. のせる (no-se-ru) 放上 ✓ ｜ 3. かう 買 ｜ 4. つかう 使用
