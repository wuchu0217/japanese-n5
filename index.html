もんだい1　よみかた

＿＿のことばの読み方を1・2・3・4からえらんでください。

1. りんごが 三つ あります。
ローマ字：san　|　中国語：三󿂃）
1.に(ni)=2　2.さん(san)=3 ✓　3.し(shi)=4　4.ご(go)=5
2. きのうは 百えんのものをかいました。
ローマ字：hyaku　|　中国語：百（100）
1.せん(sen)=1000　2.まん(man)=10000　3.ひゃく(hyaku)=100 ✓　4.じゅう(juu)=10
3. コーヒーを 少し のみます。
ローマ字：sukoshi　|　中国語：一點點（a little）
1.すこし(sukoshi) ✓　2.たくさん=many　3.ちょっと=a bit　4.あまり=not much

もんだい2　文法穴埋め

（　）に何をいれますか。1・2・3・4からえらんでください。

1. りんごが（　）あります。
数量詞の正しい形：三つ(mittsu)=3個のもの
1.三つ(mittsu) ✓　2～4は助詞不要 ✗
2. コーヒーを（　）のみます。
副詞は助詞なしで動詞を修飾する：少し(sukoshi)=a little
1.少し ✓　2～4は助詞不要 ✗
3. あの学校に学生が（　）います。
副詞はそのまま使う：たくさん(takusan)=many/a lot
1.たくさん ✓　2～4は助詞不要 ✗

もんだい3　場面

★の人は何と言いますか。1・2・3・4からえらんでください。

1. ともだちにりんご𰤥つあげたいです。★の人は何と言いますか？
五(go)=5 → 五つ(itsutsu)=5個のもの
1.五つ(itsutsu) ✓　2.百=100　3.すこし=a little　4.千=1000
2. お店でコーヒーのねだんを聞きたいです。★の人は何と言いますか？
いくら(ikura)=how much（値段を尋ねる）
1.いくら(ikura) ✓　2.なん=what　3.どこ=where　4.だれ=who
3. 先生に教室に学生が7人いることを伝えます。★の人は何と言いますか？
七(nana/shichi)=7 → 七人(shichi-nin)=7人
1.七人(shichi-nin) ✓　2.百=100　3.たくさん=many（不正確）　4.少し=a little

もんだい4　語彙

＿＿のことばとだいたいおなじいみのもの𰤡・2・3・4からえらんでください。

1. 「すこし」にいみがちかいのはどれですか？
すこし(sukoshi)=一點點 ≈ ちょっと(chotto)=a bit（どちらも少量）
1.ちょっと(chotto) ✓　2.たくさん=many　3.ぜんぶ=all　4.まったく〜ない=not at all
