📝 N5 試験モード ― 動物

もんだい１  よみかた（3問）― 漢字の読み方を選んでください

Q1. 犬 の よみかたは どれですか。

答え：いぬ (inu) ＝ 狗
①いぬ(inu)=狗 ✓  ②ねこ(neko)=貓  ③とり(tori)=鳥  ④うま(uma)=馬

Q2. 大きい の よみかたは どれですか。

答え：おおきい (ōkii) ＝ 大的
①ちいさい(chiisai)=小的  ②おおきい(ōkii)=大的 ✓  ③かわいい(kawaii)=可愛  ④はやい(hayai)=快

Q3. 魚 の よみかたは どれですか。

答え：さかな (sakana) ＝ 魚
①とり(tori)=鳥  ②うし(ushi)=牛  ③さかな(sakana)=魚 ✓  ④うま(uma)=馬

もんだい２  ぶんぽう（3問）― （　）に入るものを選んでください

Q4. わたしは 犬（　）飼っています。

答え：を (wo) ＝ 目的語助詞
「飼う」は他動詞なので目的語に「を」を使います。
①が(ga)=主語助詞  ②を(wo)=目的語助詞 ✓  ③は(wa)=主題助詞  ④に(ni)=方向・時間

Q5. この 馬は とても（　）です。

答え：大きい (ōkii)
「です」の前のい形容詞は辞書形のまま使います。
①大きい ✓  ②大きいな×(な形容詞の形)  ③大きく(副詞形)  ④大きいに×(非文)

Q6. 猫が （　） 鳴きました。

答え：大きく (ōkiku) ＝ 副詞形
動詞を修飾するにはい形容詞語幹＋「く」。大きい→大きく。
①大きい(形容詞形)  ②大きいに×  ③大きく ✓  ④大きいを×

もんだい３  ばめん（3問）― ★の人は何と言いますか

Q7. ペットショップで 小さい 犬を 見ました。★の ひとは 何と 言いますか。
★「＿＿＿＿＿」

答え：かわいいですね！(Kawaii desu ne!) ＝ 好可愛哦！
①おいしい(oishii)=好吃  ②かわいい(kawaii)=可愛 ✓  ③むずかしい(muzukashii)=困難  ④たかい(takai)=貴

Q8. 動物園で 大きい 象を 見ました。★の ひとは 何と 言いますか。
★「＿＿＿＿＿」

答え：大きいですね！(Ōkii desu ne!) ＝ 好大哦！
①かわいい(kawaii)=可愛  ②大きい(ōkii)=大 ✓  ③速い(hayai)=快  ④おいしい(oishii)=好吃

Q9. 友達に「ペットを 飼っていますか？」と 聞かれました。犬を 飼っています。★の ひとは 何と 言いますか。
★「＿＿＿＿＿」

答え：はい、犬を 飼っています。(Hai, inu wo katte imasu.) ＝ 是的，我養了一隻狗。
①いいえ、いません=不，沒有  ②はい、犬を飼っています ✓  ③はい、大きいです(不自然)  ④いいえ、かわいいです(不自然)

もんだい４  ごい（1問）― いちばん近い意味の言葉

Q10. 「犬」と いちばん ちかい なかまの ことばは どれですか。

答え：猫（ねこ, neko）＝ 貓
「犬」も「猫」も動物（どうぶつ, dōbutsu）の仲間です。
①テレビ(terebi)=電視  ②猫(neko)=貓 ✓  ③りんご(ringo)=蘋果  ④えき(eki)=車站
