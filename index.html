もんだい１ よみかた

＿＿のことばはどうよみますか。①②③④からいちばんいいものをひとつえらんでください。

1. 私の家族は五人です。
家族 = かぞく（jiā zú）家庭
① かぞく = jiā zú ✓  ② かそく ✗  ③ いえぞく ✗  ④ けぞく ✗
2. 兄は優しいです。
優しい = やさしい（wēn róu）溫柔的
① むずかしい = 困難的 ✗  ② やさしい = 溫柔的 ✓  ③ うれしい = 高興的 ✗  ④ さびしい = 寂寞的 ✗
3. 私には姉がいます。
姉 = あね（jiě jie）姊姊
① いもうと = 妹妹 ✗  ② おとうと = 弟弟 ✗  ③ あに = 哥哥 ✗  ④ あね = 姊姊 ✓

もんだい２ ぶんぽう・ぶんしょう

（　）になにをいれますか。①②③④からいちばんいいものをひとつえらんでください。

4. 私は家族（　）一緒に旅行します。
と = 和…一起（hé～yīqǐ）
① と = 和～ ✓  ② を = 受詞助詞 ✗  ③ が = 主詞助詞 ✗  ④ に = 方向・地點 ✗
5. 兄は東京に（　）います。
住んでいます = 住んで ＋ います（正在居住）
① 行って = 去 ✗  ② 住んで = 居住 ✓  ③ 来て = 來 ✗  ④ 帰って = 回去 ✗
6. 弟は（　）やさしいです。
とても ＋ い形容詞 = 非常～（fēi cháng）
① あまり = 不太（否定用）✗  ② ちょっと = 有點 ✗  ③ とても = 非常 ✓  ④ もう = 已經 ✗

もんだい３ ばめん

★の人はなんといいますか。①②③④からいちばんいいものをひとつえらんでください。

7. ★の人はお母さんを友達に紹介します。何と言いますか？
自己的家人紹介給他人時：「これは私の母です。」（這是我的媽媽。）
「お母さん」は呼びかけ・他人の母。「母（はは）」は自分の母。
8. ★の人は相手に兄弟の人数を聞きます。何と言いますか？
人数を聞く = 何人（なんにん）いますか
③「きょうだいは何人いますか」= 你有幾個兄弟姊妹？✓
9. ★の人は家族が大阪に住んでいることを伝えます。何と言いますか？
住んでいます = 正在居住（に + 住む）
② 家族は大阪に住んでいます = 家人住在大阪 ✓

もんだい４ ごい

＿＿のことばに意味がいちばんちかいものを①②③④からひとつえらんでください。

10.「家族」に意味がいちばんちかいのはどれですか？
家族（かぞく）= 家庭成員、家人
② 家の人たち = 家裡的人 ✓（最接近的意思）
